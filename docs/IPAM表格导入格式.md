# IP 管理：表格导入格式说明

本文档定义 **聚合（Aggregates）** 与 **网段（Prefixes）** 的 CSV 导入报表格式。请使用系统页面提供的「下载模板」，勿修改表头行。

## 通用约定

- 编码：**UTF-8**；模板带 **BOM**，便于 Excel 正确显示中文。
- 第一行：**固定表头**，列名与顺序必须与模板完全一致。
- 空行：忽略。
- 建议流程：**先导入聚合报表，再导入网段报表**（网段中的「所属聚合网段」依赖库中已有聚合）。

---

## 1. 聚合（Aggregates）导入

| 列名（固定顺序） | 必填 | 说明 |
|------------------|------|------|
| 网段 | 是 | CIDR，如 `10.0.0.0/8` |
| 分配机构 | 否 | 对应系统 `rir` |
| 分配日期 | 否 | `YYYY-MM-DD` |
| 描述 | 否 | |

**规则：**

- 多条聚合的 CIDR **不得彼此重叠**（含与库中已有数据）。
- 同一文件中 **同一网段（规范化后相同）出现多行** → 该行失败并记入错误列表。
- 若库中已存在相同网段：视为 **更新**；仅当对应列非空时覆盖 `分配机构`、`分配日期`、`描述`，空单元格表示保留原值。

---

## 2. 网段（Prefixes）导入

| 列名（固定顺序） | 必填 | 说明 |
|------------------|------|------|
| 网段 | 是 | CIDR |
| 状态 | 是 | `active` / `reserved` / `deprecated` / `container`（不区分大小写） |
| 描述 | 否 | |
| 地址池 | 否 | `是`/`否`、`true`/`false`、`1`/`0` |
| 标记已用 | 否 | 同上 |
| VLAN | 否 | 整数；空表示不填 |
| 位置 | 否 | |
| 所属聚合网段 | 否 | 填写聚合的 CIDR，系统解析为所属聚合 |
| 聚合ID | 否 | 数字 ID；**若与「所属聚合网段」同时填写，以聚合ID为准** |

**规则：**

- 同一文件中 **同一网段（规范化后相同）出现多行** → 该行失败。
- 若库中已存在相同网段：**更新**；非空列覆盖对应字段，布尔/枚举按解析规则处理。
- `aggregate_id` / 所属聚合网段 与线上一致：若指定聚合，网段须落在该聚合 CIDR 内。

**状态中文对照（填写仍用英文枚举值）：**

| 值 | 含义 |
|----|------|
| active | 在用 |
| reserved | 保留 |
| deprecated | 废弃 |
| container | 容器 |

---

## 3. 与 PRD 的关系

本功能为 [PRD-IP管理功能.md](./PRD-IP管理功能.md) 中 IP 管理模块的增量能力；数据仍写入 `ipam_aggregates`、`ipam_prefixes`，与 DHCP 关联方式不变。
